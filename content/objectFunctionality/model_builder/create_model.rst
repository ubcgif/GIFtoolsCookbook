.. _objectFunctionalityMBbuild_surf:

Create Model Using Surfaces
---------------------------

This functionality allows the user to set the physical property values for cells that lie between two surfaces. The surfaces may be defined by topography (TOPOdata) or surfaces (SURFACEdata). Physical property values can be set for cells within the active or inactive space. This functionality is accessed through:

**Create model** |rarr| **Using surfaces**
