Run a Forward Model
===================

Prior to running the forward model, the user must :ref:`write <fwdWriteAll>` the necessary input files to the working directory. For the forward model, that includes the input file, mesh, physical property model and possibly topography. Click on the forward modeling item, select the menu showing its class (e.g., ``E3Dforward``):

**[Forward modeling class]** |rarr| **Run [forward model name]**
