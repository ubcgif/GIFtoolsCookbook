Set Data for OcTree Mesh Generation
===================================

For generating OcTree meshes with Fortran utilities, we must set an appropriate data object. For example, if E3DMT OcTree mesh generation is used, an *IMPdata*, *APPdata* or *ZTEMdata* object must be used. For any Fortran Octree mesh generation utility, this functionality is accessed through:

**[Utility class]** |rarr| **Set data object**
