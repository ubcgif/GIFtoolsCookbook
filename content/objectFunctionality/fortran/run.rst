Run a utility program
=====================

The user choose the menu item **Run** to run the utility. GIFtools will write out any files that are required to run the utility (unlike an inversion). Click on the utility item, select the menu showing its class (e.g., ``Interpolate Model utility``):

**[Utility class]** |rarr| **Run utility**
