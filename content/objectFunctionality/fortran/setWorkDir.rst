Set the utility's working directory
===================================

The working directory tells GIFtools where to output files (and run the utility) associated with the selected item. Click on the utility item, select the menu showing its class (e.g., ``Interpolate model utility``):

**[Utility class]** |rarr| **Set working directory**

.. figure:: ../../../images/utilityMenu.png
    :align: center
    :width: 400

**NOTE:** The shortcut for the functionality is ``control + d``.
