Set the equivalent source working directory
===========================================

The working directory tells GIFtools where to output files (and run the program) associated with the selected item. It is suggested to set the directory *prior to setting any other properties* . Click on the equivalent-source item, select the menu showing its class (e.g., ``MAGEquivalent source`` or ``GRAVEquivalent source``):

**[Equivalent-source class]** |rarr| **Set working directory**

.. figure:: ../../../../images/esrcSimpleMenu.png
    :align: center
    :width: 400

**NOTE:** The shortcut for the functionality is ``control + d``.
